This CapabilityStatement describes the expected capabilities of a client that is capable of responding a Da Vinci Notification with a follow up FHIR RESTful query to the Sender. These expectations are the same as described in the HL7 FHIR® US Core Implementation Guide STU3 Release 3.1.0 CapabiltyStatement for the US Core Client
