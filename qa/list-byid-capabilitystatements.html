• Notification Receiver CapabilityStatement

    This CapabilityStatement describes the expected capabilities of a server that is capable of receiving a Da Vinci Unsolicited Notification transacted with the $process-message operation.

  • Notification Sender CapabilityStatement

    This CapabilityStatement describes the expected capabilities of a client that is capable of sending a Da Vinci Unsolicited Notification transacted with the $process-message operation.

  • Query Requester CapabilityStatement

    This CapabilityStatement describes the expected capabilities of a client that is capable of responding a Da Vinci Notification with a follow up FHIR RESTful query to the Sender. These expectations are the same as described in the HL7 FHIR® US Core Implementation Guide STU3 Release 3.1.0 CapabiltyStatement for the US Core Client

  • Query Responder CapabilityStatement

    This CapabilityStatement describes the expected capabilities of a server that is capable of responding to a follow up FHIR RESTful query from an Intermediary/Recipient. These Expectations are the same as described in the HL7 FHIR® US Core Implementation Guide STU3 Release 3.1.0 CapabiltyStatement for the US Core Server
